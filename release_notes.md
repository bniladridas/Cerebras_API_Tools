## Installation
```bash
# Install dependencies
brew install curl nlohmann-json

# Compile
g++ -std=c++11 -o cli cli.cpp -lcurl -I/opt/homebrew/include

# Run
./cli
```

## Configuration
Create a .env file with your API key:
```
CEREBRAS_API_KEY=your-api-key-here
```
